## Push commands (run in VS Code terminal, one at a time):
```
git init
git add .
git commit -m "Launch portfolio site"
git branch -M main
git remote add origin https://github.com/watkinsamya/watkinsamya.github.io.git
git push -u origin main --force
```

Then go to: github.com/watkinsamya/watkinsamya.github.io → Settings → Pages
→ Branch: main, folder: / (root) → Save

Your site will be live at https://watkinsamya.github.io in ~2 minutes.
